打开文件：

使用命令 vim filename 打开一个文件。
保存和退出：

按下 Esc 键进入命令模式。
输入 :w 并按下回车键保存文件。
输入 :q 并按下回车键退出Vim。
输入 :wq 并按下回车键保存并退出Vim。
光标移动：

使用方向键或 h、j、k、l 键进行光标移动。
使用 gg 快捷键移动到文件开头。
使用 G 快捷键移动到文件末尾。
文本编辑：

在命令模式下，按下 i 键进入插入模式，开始编辑文本。
在插入模式下，按下 Esc 键返回命令模式。
复制、剪切和粘贴：

在命令模式下，将光标移动到要复制或剪切的文本位置。
按下 v 进入可视模式。
使用方向键或 h、j、k、l 键选择文本。
按下 y 复制选中文本，按下 d 剪切选中文本。
将光标移动到要粘贴的位置，在命令模式下按下 p 粘贴。
搜索和替换：

在命令模式下，输入 /search_term 搜索指定字符串。
按下 n 键查找下一个匹配项。
按下 :%s/search_term/replace_term/g 替换文本中的所有匹配项。
撤销和重做：

在命令模式下，按下 u 撤销最近的编辑操作。
按下 Ctrl + r 重做上一次撤销的操作。

更多操作
若要在Vim中执行Shell命令，请按下 : 进入命令模式，然后输入 !command。
若要分割窗口，请按下 Ctrl + w，然后按下 v 分割为垂直窗口，或者按下 s 分割为水平窗口。
若要在不保存退出Vim时强制退出，请在退出命令后添加 !，例如 :q!。


复制、剪切和粘贴：

在可视模式下，你可以使用 y 命令复制选中的文本，使用 d 命令剪切选中的文本，然后使用 p 命令粘贴。
快速移动：

使用 Ctrl + f 快捷键向下滚动一页。
使用 Ctrl + b 快捷键向上滚动一页。
使用 Ctrl + d 快捷键向下滚动半页。
使用 Ctrl + u 快捷键向上滚动半页。
使用 H 快捷键将光标移动到屏幕顶部。
使用 M 快捷键将光标移动到屏幕中间。
使用 L 快捷键将光标移动到屏幕底部。
文本搜索：

在命令模式下，你可以使用 / 进行文本搜索，并按下 Enter 键查找下一个匹配项。
多窗口操作：

使用 :sp filename 命令将当前窗口分割为上下两个窗口，并在新窗口打开指定文件。
使用 :vs filename 命令将当前窗口分割为左右两个窗口，并在新窗口打开指定文件。
设置行号：

在命令模式下，使用 :set number 命令显示行号。
使用 :set nonumber 命令隐藏行号。
